Penugasan 4 — Framework & API

Deskripsi

Pada penugasan ini, saya mengeksplorasi penggunaan framework modern untuk pengembangan web serta integrasi dengan API eksternal untuk mengambil dan menampilkan data secara dinamis. Penugasan ini memperdalam pemahaman tentang arsitektur aplikasi web modern.

Tujuan

Mengintegrasikan framework front-end dan API untuk membangun aplikasi web yang dinamis, responsif, dan terstruktur menggunakan praktik pengembangan modern.

Hasil

Berhasil mengimplementasikan aplikasi web menggunakan framework Vue.js dengan integrasi REST API. Aplikasi mampu mengambil data dari server, menampilkan secara dinamis, dan memberikan pengalaman pengguna yang responsif dan interaktif.
