Penugasan 3 — JavaScript Interaktif

Deskripsi

Penugasan ini berfokus pada penerapan JavaScript untuk membuat elemen interaktif pada halaman web. Materi meliputi manipulasi DOM, event handling, dan validasi form untuk meningkatkan user experience.

Tujuan

Mengimplementasikan logika interaktif pada halaman web menggunakan JavaScript, termasuk pengelolaan event, manipulasi elemen DOM, dan pembuatan fitur yang meningkatkan interaktivitas pengguna.

Hasil

Berhasil membuat fitur interaktif seperti validasi form real-time, animasi berbasis scroll, manipulasi DOM dinamis, serta implementasi event listener yang efisien untuk berbagai jenis interaksi pengguna.
